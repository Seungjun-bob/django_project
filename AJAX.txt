[ AJAX ]
- 웹 클라이언트에서 웹 서버에 요청하는 통신 방법 중 하나
- 웹 클라에서 서버에 요청 보내는 방법
  (1) 직접 URL 문자열을 입력하여 요청하기 ( view 함수에 있는 문자열 )
  (2) <a> 태그를 사용하여 요청하기
  (3) <form> 태그를 사용하여 요청하기
  (4) 자바스크립트 코드(location.href)로 요청
  --> 현재 페이지를 없애고 요청 페이지로 이동

- AJAX 기술을 사용하여 클라에서 서버에 요청하는 방법
  - JavaScript 코드로 구현 ( jQuery 라이브러리를 사용하면 구현 방법이 간단해짐 )
  - 페이지 이동이 일어나지 않는다.
  - 데이터 전송량이 많이 축소되므로 모바일 환경에서 적합하다.
  - HTML, XML, JSON, ... 응답형식에 제한이 없다.

[ AJAX 기술의 구현 방법 ]

(1) XMLHttpRequest 객체를 생성한다.
	let req = new XMLHttpRequest()"
(2) AJAX 기술로 서버에 요청하고 응답이 왔을 때 수행할 코드를 함수로 만들어서 
    load 이벤트에 대한 핸들러 등록한다. (이벤트가 발생했을 때 수행되는 코드를 담고 있는 함수)
	req.onload = 함수
(3) AJAX 통신으로 요청하려는 서버 프로그램의 URL 문자열을 가지고정보를 작선한다.
	req.open('요청방식', 'URL', true)  ex) 택배 송장을 작성하는 작업과 비슷
(4) AJAX 통신 요청을 서버에 보낸다.
	req.send()

[ 함수 구현 ]
AJAX 통신에 대한 응답이 왔을 때 수행하게 되는 기능을 구현한다.
서버로 부터 응답되는 형식(일반 텍스트, XML, JSON, 이미지, ...)에 따라 구현 방법이 달라진다.

	req.responseText 를 사용해서 응답된 내용을 읽는다.
	JSON 응답 -> JSON.parse()를 사용해서 JavaScript 객체로 생성하여 사용한다.
